## 2. 権限リセット問題

### 2.1 ATT
- [ ] A: アプリ起動 → ATTダイアログは出ない (オンボ中)
- [ ] A: オンボ完了 → ホーム到達 → 1.5秒後にATTダイアログ
- [ ] A: 拒否 → 設定 > 権限ステータスで「拒否」表示
- [ ] A: アプリ再起動 → ATTダイアログは **出ない** (永続化確認)

### 2.2 マイク
- [ ] 言葉を追加 > 録音ボタン → 初回はダイアログ
- [ ] アプリ再起動 → 録音ボタン押すとダイアログは出ない (権限保持)

### 2.3 通知
- [ ] オンボ完了時に通知許可ダイアログ
- [ ] 拒否してもアプリは動く
- [ ] 再起動で再ダイアログ出ない

### 2.4 シミュレータ削除→再インストール
- [ ] `xcrun simctl uninstall ... com.mendoi.AFFARMENTNEMO`
- [ ] 再インストール → ATTダイアログがオンボ完了後に再度出る (これは仕様通り、実機でもアプリ削除すれば同じ)
